自动将目录下的文件全部上传到alist的指定路径

# 运行
先配置setting.py文件
```python
# 安装依赖项
pip install -r requirements.txt

# 运行
python main.py
```

# 多配置文件运行
运行命令`python main_multi.py`,程序自动检测目录configs下所有`yaml`和`py`文件，根据提示选择使用哪个配置文件或全部运行
